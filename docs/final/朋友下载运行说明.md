# 朋友下载运行说明

> 给非开发者的朋友看的：下载项目后如何在自己的电脑上运行。

## 前提条件

- 有一个 DeepSeek API Key（在 [platform.deepseek.com](https://platform.deepseek.com) 注册获取）
- 电脑上已安装 Python 3.10+ 和 Node.js 18+

## 快速开始（7 步）

### 第 1 步：启动后端

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --host 127.0.0.1 --port 8000
```

后端启动后保持终端窗口不要关闭。

### 第 2 步：打开前端

在浏览器打开 `frontend-demo/index.html`，或使用 Live Server。

### 第 3 步：进入设置页

点击左侧导航栏的 **系统设置**。

### 第 4 步：填写 DeepSeek API Key

在 **DeepSeek 模型配置** 卡片中：
- Base URL 保持默认 `https://api.deepseek.com`
- 填入自己的 **DeepSeek API Key**（格式：`sk-xxxxxxxx`）
- 模型选择 **deepseek-v4-pro**（推荐）

### 第 5 步：保存并测试

1. 点击 **保存配置**
2. 点击 **测试连接**
3. 确认显示 "DeepSeek V4 Pro 连接成功"

### 第 6 步：开始演示

返回首页，点击 **开始 AI 学习演示**。

### 第 7 步（可选）：配置星火

如果需要使用讯飞星火作为备用模型，展开 **讯飞星火（可选配置）** 填写即可。

## 常见问题

### Q: 我没有 DeepSeek API Key 怎么办？
A: 注册 DeepSeek 开放平台：https://platform.deepseek.com ，新用户有免费额度。

### Q: deepseek-v4-pro 连接失败？
A: 可能是账户权限不足。可以尝试改用以下模型：
- `deepseek-v4-flash`（更快更便宜）
- `deepseek-chat`（旧版兼容）

在设置页的模型下拉框中选择切换即可。

### Q: GitHub 上为什么没有 API Key？
A: 这是**正常的**。API Key 保存在本机 `backend/.env` 文件中，该文件已加入 `.gitignore`，不会被提交到 GitHub。每个人需要自己填写。

### Q: API Key 保存在哪里？
A: 保存在项目目录下的 `backend/.env` 文件。前端页面的 API Key 输入框类型为 `password`，不会明文展示。后端日志也不会打印 API Key。

## 安全说明

- API Key 仅保存在本机 `backend/.env`
- `backend/.env` 不会提交到 GitHub（已加入 `.gitignore`）
- 页面不展示完整 Key（password 输入框）
- 后端日志不打印 Key
- 前端不存储 Key 到 localStorage
