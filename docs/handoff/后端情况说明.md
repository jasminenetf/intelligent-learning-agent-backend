# 后端情况说明

## 项目名称
高等教育个性化学习资源多智能体系统后端

## 当前阶段
已完成阶段 0 到阶段 6。

## 当前后端能力
1. 用户注册、登录、JWT 鉴权（student/teacher/admin）
2. 课程创建与资料上传
3. PDF/DOCX/TXT 基础解析与文本切分
4. ChromaDB 本地向量索引
5. RAG 检索（按 course_id 过滤）
6. LLM Provider 抽象层（Mock/Spark/DeepSeek）
7. RAG QA 问答闭环（检索→prompt→LLM→answer+citations）
8. LangGraph 5 Agent 多智能体教学流程

- agent_trace 完整追踪 5 步
