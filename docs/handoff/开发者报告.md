## 一、项目概述

本项目是面向高校学生的个性化学习资源生成与辅导系统后端。学生通过对话提问，系统基于课程知识库检索相关内容，调用大语言模型生成带引用的教学回答，并支持多智能体协作分析学生画像、审核答案质量。

**核心差异点：**
- 不是简单的 RAG 问答，而是 LangGraph 5 Agent 教学流程
- 不是固定 LLM，而是可插拔 Provider 架构（Mock/Spark/DeepSeek）
- 不需要前端也能独立运行，20 个 REST API 全部可用

## 三、技术栈

| 层 | 技术 | 作用 |
|----|------|------|
| Web 框架 | FastAPI | API 路由、依赖注入 |
| ORM | SQLModel | SQLite 数据库（users/courses/files/chunks） |
| 认证 | PyJWT + argon2 | JWT token，密码哈希 |
| 向量库 | ChromaDB | 本地持久化向量检索 |
| 文档解析 | PyMuPDF + python-docx | PDF、DOCX 文本提取 |
| 文本切分 | 纯 Python chunker | 800/120 字符重叠切分 |
| LLM 调用 | openai SDK | 统一接口 (Spark/DeepSeek) |
| Agent 编排 | LangGraph | 5节点 DAG 教学流程 |
| 部署 | Docker Compose | Dockerfile + compose.yml 已就绪 |
