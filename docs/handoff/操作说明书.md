# 智能学习Agent 后端 — 操作说明书

> 适用读者：接手开发的开发者、运维人员、评审老师

---

## 环境要求

- Python 3.10+
- WSL2 / Linux / macOS
- 可选：Docker 24+（部署使用）

## 快速启动

```bash
# 1. 进入项目
cd /home/zhang/projects/intelligent-learning-agent

# 2. 激活虚拟环境
source .venv/bin/activate

# 3. 安装依赖
pip install -r backend/requirements.txt

# 4. 启动服务
uvicorn app.main:app --host 0.0.0.0 --port 8000 --app-dir backend

# 5. 验证
curl http://127.0.0.1:8000/health
curl http://127.0.0.1:8000/api/version
```

## API 文档

启动后访问：http://127.0.0.1:8000/docs (Swagger UI 交互式文档)

## 快速测试流程

### 1. 注册教师
```bash
curl -X POST http://127.0.0.1:8000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"teacher1","email":"t@x.com","password":"12345678","role":"teacher"}'
```

### 2. 登录获取 Token
```bash
TOKEN=$(curl -s -X POST http://127.0.0.1:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"teacher1","password":"12345678"}' \
  | python3 -c "import sys,json; print(json.load(sys.stdin)['access_token'])")
```

### 3. 创建课程
```bash
curl -X POST http://127.0.0.1:8000/api/courses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name":"高等数学","description":"测试课程"}'
```

### 4. 上传教材
```bash
curl -X POST http://127.0.0.1:8000/api/courses/1/files \
  -H "Authorization: Bearer $TOKEN" \
  -F "file=@高数上_demo_knowledge.txt"
```

### 5. 构建 RAG 索引
```bash
curl -X POST http://127.0.0.1:8000/api/rag/courses/1/build \
  -H "Authorization: Bearer $TOKEN"
```

### 6. RAG 问答
```bash
curl -X POST http://127.0.0.1:8000/api/qa/courses/1/ask \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"question":"导数是什么？","top_k":5}'
```

### 7. 多智能体教学
```bash
curl -X POST http://127.0.0.1:8000/api/agents/course/1/tutor \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"question":"导数是什么？用简单语言讲。","top_k":5}'
```

## 配置说明

复制 `.env.example` 为 `.env` 并填入实际值：

```bash
cp .env.example .env
```

关键配置项：
| 变量 | 说明 | 必填 |
|------|------|------|
| JWT_SECRET_KEY | JWT 签名密钥 | 生产必填 |
| SPARK_API_KEY | 讯飞星火 API Key | 使用 Spark 时 |
| DEEPSEEK_API_KEY | DeepSeek API Key | 使用 DeepSeek 时 |
| LLM_PROVIDER | mock/spark/deepseek | 默认 mock |

## Docker 部署（Docker 环境就绪后）

```bash
# 安装 Docker
sudo apt install -y docker.io docker-compose-v2
sudo usermod -aG docker $USER

# 启动
docker compose up -d --build

# 查看日志
docker compose logs -f backend

# 停止
docker compose down
```

## 故障排查

| 问题 | 解决 |
|------|------|
| `ModuleNotFoundError` | 检查 venv 激活：`source .venv/bin/activate` |
| Spy API 调用失败 | 检查 SPARK_API_KEY 是否正确 |
| ChromaDB 无法写入 | 检查 `data/chroma/` 目录权限 |
| 检索无结果 | 先执行 `/api/rag/courses/{id}/build` |
| 端口占用 | `kill $(lsof -ti:8000)` |
