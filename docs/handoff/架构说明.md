## 核心数据流

### 流程 1：课程知识库构建
```
教师上传 PDF/TXT → file_storage 安全存储
  → document_parser 提取文本 → chunker 切分(800/120)
  → knowledge_chunks 入库 → ChromaDB build index
```

### 流程 2：基础 RAG 问答
```
学生提问 → [POST /api/qa/courses/{id}/ask]
  → search_course(question, top_k) → ChromaDB 检索
  → build_rag_prompt(chunks, profile) → LLM generate
  → answer + citations 返回
```

### 流程 3：5 Agent 多智能体教学
```
学生提问 → [POST /api/agents/course/{id}/tutor]
  → supervisor_node: 入口，clamp top_k≤10
  → profile_node: 规则抽取学生画像(5维)
  → rag_node: ChromaDB 检索→citations
  → lecture_node: build prompt + LLM生成
  → verifier_node: 审核 4 项(非空/引用/来源)
  → student_profile + verified_answer + agent_trace 返回
```

## LLM Provider 抽象层

```
                 ┌─────────────────┐
                 │  get_llm_p...()  │
                 └────────┬────────┘
                          │
          ┌───────────────┼───────────────┐
          ▼               ▼               ▼
┌─────────────────┐ ┌───────────┐ ┌─────────────┐
│ MockLLMProvider │ │ Spark     │ │ DeepSeek    │
│ (always works)  │ │ (需 Key)  │ │ (需 Key)    │
└─────────────────┘ └───────────┘ └─────────────┘
                          │               │
                    SPARK_API_KEY   DEEPSEEK_API_KEY
                    缺失→fallback    缺失→fallback
                    到 Mock          到 Mock
```

选择规则:
- `LLM_PROVIDER=mock` → MockLLMProvider
- `LLM_PROVIDER=spark + SPARK_API_KEY` → SparkLLMProvider
- `LLM_PROVIDER=deepseek + DEEPSEEK_API_KEY` → DeepSeekProvider
- Key 缺失 → 自动 fallback Mock（不崩溃）
- **不自动在 Spark/DeepSeek 间跳转**

## 学生画像维度

| 维度 | 提取方式 | 枚举值 |
|------|----------|--------|
| knowledge_level | 关键词检测 | beginner/intermediate/advanced |
| cognitive_style | 关键词检测 | conceptual/logical/practice_oriented |
| needs_examples | 推导 | true/false |
| needs_step_by_step | 推导 | true/false |
| detected_weakness | 预留 | 空字符串 |

## 权限模型

| 角色 | 权限 |
|------|------|
| student | 注册/登录，QA 问答，Agent tutor |
| teacher | student 全部 + 创建课程 + 上传文件 + 构建 RAG + LLM 管理 |
| admin | teacher 全部 |

## 关键配置

| 变量 | 默认值 | 说明 |
|------|--------|------|
| DATABASE_URL | sqlite:///./data/app.db | SQLite 路径 |
| CHROMA_PERSIST_DIR | ./data/chroma | ChromaDB 持久化目录 |
| EMBEDDING_PROVIDER | hash_mock | 可选 sentence_transformers |
| LLM_PROVIDER | mock | 可选 spark/deepseek |
| JWT_SECRET_KEY | change-me-in-production | 生产环境必须更换 |

## 文件安全

- `sanitize_filename()` — 剥离路径穿越字符
- 仅允许 .pdf / .docx / .txt
- 上传上限 30MB
- 存储到 `data/raw/{course_id}/`
- 不进入 Git
