# 答辩演示脚本 — 完整版（5-8分钟）

> 按步骤操作，每步含操作、页面位置、讲解词、备用方案

---

## 演示前准备（1分钟前）

```bash
# 终端1: 启动后端
cd /home/zhang/projects/intelligent-learning-agent/backend
source ../.venv/bin/activate
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000

# 终端2: 启动前端
cd /home/zhang/projects/intelligent-learning-agent/frontend-demo
python -m http.server 5173
```

**检查**: 浏览器打开 http://127.0.0.1:5173

---

## Step 1: 开场（30秒）

**操作**: 展示封面幻灯或系统标题

**讲解词**: "大家好，我们展示的是高等教育个性化学习资源多智能体系统。它基于 DeepSeek 真大模型，为每位学生自动生成个性化的学习资源。"

---

## Step 2: 检查系统状态（30秒）

**操作**: 点击"检查状态"按钮

**页面位置**: 左侧面板 → 系统状态区

**讲解词**: "我们先检查系统状态。可以看到 LLM 使用 DeepSeek-chat 真模型，Embedding 使用 sentence-transformers，不是 Mock。"

**预期输出**: 🟢 deepseek/deepseek-chat / Embedding:sentence_transformers

**备用**: 如连接失败，检查后端是否启动；如显示 mock，检查 backend/.env 配置

---

## Step 3: 登录（15秒）

**操作**: 点击"登录"按钮（已预填用户名密码）

**讲解词**: "系统使用 JWT 认证，登录后 Token 自动填入，后续所有 API 自动携带。"

**备用**: 如提示用户已存在，点击"注册"后再登录

---

## Step 4: 提取学生画像（45秒）

**操作**: 点击"提取画像"

**页面位置**: 左侧 → 学生画像区

**讲解词**: "系统支持自然语言描述提取画像。这位同学说自己是计算机专业，导数和极限比较薄弱，喜欢用图和例题学习，节奏慢。我们点击提取画像。"

**预期输出**: knowledge_level=beginner, cognitive_style=conceptual, weak_points=["导数","极限"], pace_preference=slow

**讲解词**: "系统识别出他是初学者水平，概念型认知风格，薄弱点是导数和极限，偏好慢节奏和可视化资源。8维画像存入数据库，后续资源生成会参考这些信息。"

**备用**: 如提取失败，展示"查看画像"显示默认画像

---

## Step 5: RAG 问答（45秒）

**操作**: 点击"提问"（问题已预填："根据课程资料解释导数和函数变化率的关系"）

**讲解词**: "在资源生成前，我们先测试 RAG 问答。系统从课程知识库检索相关资料，结合 DeepSeek 生成回答。注意返回结果包含引用来源，每个回答都标注了来自哪份资料、第几页。"

**预期输出**: 回答内容 + citations(source, page_number, score)

**备用**: 如无资料，使用话题"函数基本概念"

---

## Step 6: 生成学习路径（45秒）

**操作**: 清空资源类型选择，只选 study_plan，点击"生成资源"

**讲解词**: "我们先看学习路径规划。输入主题'导数与极限入门'，选择 study_plan。"

**预期输出**: 4步学习路径卡片，每步含原因、资源推荐、预计时间、练习建议

**讲解词**: "系统生成了4步学习路径：先学基础概念→再学关键方法→然后综合应用→最后查漏补缺。每步结合了学生画像——因为学生是视觉型，第1步推荐了 mindmap；节奏慢，每步预估30分钟左右。后续还推荐了下个学习主题。"

---

## Step 7: 生成思维导图（30秒）

**操作**: 只选 mindmap → 生成资源

**讲解词**: "然后是思维导图。系统生成 Mermaid 格式的思维导图，前端用 Mermaid.js 渲染。可以看到它把导数与极限分为4大分支：极限思想、导数概念、函数基础、积分初步，每个分支根据课程资料和画像展开子节点。"

---

## Step 8: 生成测验题（30秒）

**操作**: 只选 quiz → 生成资源

**讲解词**: "测验题自动生成。7道选择题，包含基础题和应用题，每道题有解析。正确选项高亮显示。题目结合了学生的薄弱知识点，会多出导数相关题目。"

---

## Step 9: 生成 PPT 并下载（30秒）

**操作**: 只选 ppt → 生成资源 → 点击"下载 PPTX"

**讲解词**: "PPT 课件一键生成。8页标准结构：封面、学习目标、核心概念、例题、误区、练习建议、总结。内容由 DeepSeek 根据课程资料生成，可以下载 PPTX 文件。"

**备用**: 如浏览器拦截下载，右键另存为

---

## Step 10: 一键演示（30秒）

**操作**: 点击"全流程演示"

**讲解词**: "最后我们演示一键全流程。点击全流程演示，系统依次检查状态、提取画像、RAG问答、生成全部5类资源。整个过程约30秒。"

---

## Step 11: 总结（30秒）

**讲解词**: "总结一下，我们的系统实现了三个闭环：
1. 知识库闭环：教材→OCR→向量化→ChromaDB→RAG检索
2. 个性化闭环：自然语言→画像→DeepSeek→5类资源→路径规划
3. 防幻觉闭环：检索→生成→校验→引用标注

全链路使用真 LLM 和真 Embedding，支持 OpenAI 兼容接口可以对接 LobeChat。谢谢！"
