# 最终答辩录屏脚本 — 高等教育个性化学习资源多智能体系统

---

## 3 分钟版录屏脚本

### 0:00-0:20 项目介绍

> "这是'高等教育个性化学习资源多智能体系统'——一个基于多智能体协同架构的 AI 学习工作台。它利用课程资料知识库、学生画像和多个 AI 智能体的协作，为学生生成个性化问答、思维导图、测验、讲义和学习路径。"

点击路径：打开浏览器 → http://127.0.0.1:5173

### 0:20-0:45 演示账号与课程

> "系统默认配置了演示账号和高等数学课程，包含 16 个课程知识点，知识库已就绪。"

点击路径：点击"进入演示学习环境" → 自动进入学习助手
展示重点：顶部课程名"高等数学上"、知识库就绪标签

### 0:45-1:20 智能问答与 RAG 引用

> "我们点击示例问题'导数和函数变化率有什么关系？'系统会通过多智能体协同——Tutor Agent 拆解问题、Informer Agent 检索课程资料、Verifier Agent 交叉验证答案，最终生成回答并附带文献引用。"

点击路径：点击示例问题 → 观察 loading → Agent 状态变化 → 回答出现
展示重点：回答内容、右侧文献溯源卡片、Agent 状态流转

### 1:20-1:50 Agent Trace 与 Citation 高亮

> "右侧可以看到多智能体协作过程：AI 学习助手、资料检索、内容校验、资源生成——每个 Agent 的实时状态。点击引用编号，右侧对应来源卡片会高亮显示，实现学术引用溯源闭环。"

点击路径：点击 [1] citation badge → 右侧卡片高亮闪烁
展示重点：Agent 呼吸灯动画、citation 高亮联动

### 1:50-2:25 思维导图与测验

> "基于当前问题，我们可以一键生成思维导图。导图默认展示图形，源码折叠，支持缩放和拖拽。切换到练习题库，每道题可以点击作答，系统即时反馈正确/错误，并统计正确率。答错时还会提示相关知识点薄弱，建议回到学习路径复习。"

点击路径：点击"生成思维导图" → 展示缩放 → 点击"启动测验" → 点击答题
展示重点：思维导图缩放/拖拽、测验选项绿/红反馈、正确率统计

### 2:25-2:45 PPT 下载与学习路径

> "生成 PPT 后可以直接下载 PowerPoint 文件。学习路径展示了个性化的学习步骤，包括每个步骤的主题、预计时间和练习建议。"

点击路径：点击"生成 PPT" → 展示下载按钮 → 切换到学习路径
展示重点：PPT 下载按钮、学习路径时间线卡片

### 2:45-3:00 总结

> "系统实现了从课程资料上传、知识库构建、学生画像提取、到多智能体协同生成个性化学习资源的完整闭环。技术栈包括 FastAPI + LangGraph + ChromaDB + DeepSeek，所有代码开源在 GitHub。"

## 5 分钟版录屏脚本

### 0:00-0:30 项目定位

> "这是一个面向高等教育的个性化学习资源多智能体系统。传统学习平台只能提供静态课件，而我们的系统通过多智能体协同——Tutor、Informer、Verifier、Practice 四个 Agent 的分工协作——为学生动态生成个性化、可溯源的学习资源。"

### 0:30-1:00 系统入口与课程资料

> "登录演示账号后，系统自动选择了高等数学上课程，课程已有 16 个知识点切片的向量化知识库。右侧数据看板可以看到 AI 模型已连接、课程资料已就绪。"

点击路径：演示登录 → 展示数据看板 → 展示课程列表和知识库状态

### 1:00-1:50 RAG 问答与引用

> "在学习助手页面提问。系统通过 RAG 检索增强生成技术，从课程资料中检索相关知识，再由大模型生成回答。每条回答都附带文献引用，点击引用编号可以高亮对应的来源卡片，确保学术透明。"

点击路径：提问 → 展示回答 → 点击 citation

### 1:50-2:30 多智能体过程

> "右侧实时展示多智能体协作过程。从 Tutor Agent 拆解问题、Profile Agent 分析学生画像、Informer Agent 检索资料、Verifier Agent 验证答案，到 Practice Agent 生成资源建议——每个 Agent 的状态都是动态变化的，让学生看到 AI 不是'黑箱'。"

展示重点：Agent 卡片依次激活，呼吸灯动画

### 2:30-3:20 Artifacts 沙盒

> "中间 Artifacts 工作区是独立的资源沙盒。可以生成思维导图——支持缩放、平移、源码查看，测验题——支持点击作答和即时反馈，讲义文档——Markdown 渲染的教学内容。所有资源都基于当前问题和学生画像个性化生成。"

点击路径：生成导图 → 缩放 → 生成测验 → 答题 → 生成讲义

### 3:20-4:00 PPT 下载与学习路径

> "PPT 课件可以下载为标准的 PowerPoint 文件，方便线下复习。学习路径展示了基于学生画像的个性化学习路线，包括每步的主题、推荐资源和预计时间。"

点击路径：生成 PPT → 下载 → 展示学习路径

### 4:00-4:30 画像雷达图

> "右侧的学生画像雷达图展示了 8 维学习特征：知识基础、认知风格、学习目标、学习节奏等。每次问答后画像会动态更新，让系统越来越了解学生。"

展示重点：雷达图、画像标签

### 4:30-5:00 总结

> "系统完整覆盖了赛题要求的全部核心功能：多智能体协同架构、课程资料知识库、RAG 防幻觉问答、5 类个性化资源生成、学生画像动态更新。技术方案采用 LangGraph 图引擎编排多智能体流程，前端纯 HTML/CSS/JS 实现沉浸式工作台，具备演示和答辩的完整产品形态。"

## 风险提示与应急方案

| 风险 | 应对 |
|------|------|
| DeepSeek API 慢 | 前端 25s 后自动显示演示答案 fallback |
| PPT 下载慢 | 录屏前提前生成一次，第二次有缓存 |
| 浏览器异常 | 刷新页面 http://127.0.0.1:5173 |
| 后端崩溃 | `bash scripts/start_app.sh` 重启 |
| 录屏时网络波动 | 演示答案模式可离线展示 |
