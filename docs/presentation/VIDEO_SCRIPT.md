# 录屏脚本 — 3-5分钟演示视频

---

## 录屏前检查清单

- [ ] 所有终端启动完毕（后端8000 + 前端5173）
- [ ] 浏览器已打开 http://127.0.0.1:5173
- [ ] 录屏软件已准备（OBS/系统自带）
- [ ] 麦克风测试正常
- [ ] 桌面通知已关闭
- [ ] 网速正常（DeepSeek API 需联网）
- [ ] 后台无敏感信息（Key、密码等）

---

## 片头（10秒）

**旁白**: "高等教育个性化学习资源多智能体系统演示"

## 痛点引出（20秒）

**旁白**: "大学生基础差异大、教师难以准备个性化资源、学生不知知识短板。我们的系统解决这些问题。"

## 系统状态检查（15秒）

**画面**: 前端页面，点击"检查状态"

**旁白**: "系统基于 DeepSeek 真大模型和 sentence-transformers 真嵌入。不是 Mock。"

## 登录与画像提取（30秒）

**画面**: 依次点击 登录 → 提取画像

**旁白**: "登录后输入自然语言描述，系统用 DeepSeek 提取8维学生画像。这位同学导数极限薄弱、偏好可视化学习、节奏较慢。"

**关键截图**: 8维画像字段展示

---

## RAG 问答（25秒）

**画面**: 点击"提问"，展示回答+引用

**旁白**: "RAG 问答基于课程资料检索，每句回答标注引用来源。不是凭空生成。"

**关键截图**: 回答 + citations

---

## 5类资源生成（90秒）

**画面**: 依次生成 study_plan → mindmap → quiz → ppt（每类约20秒）

**旁白**: 
- "学习路径：4步计划，结合画像和课程资料"
- "思维导图：Mermaid 自动渲染，分支结构清晰"
- "测验题：7道选择题，自动生成解析"
- "PPT课件：8页，一键下载 PPTX"

**关键截图**: 每类资源卡片

---

## 技术亮点（20秒）

**画面**: 架构图淡入

**旁白**: "三大闭环：知识库 OCR→RAG、个性化画像→生成、防幻觉检索→验证→引用。全链路真 LLM。"

## 建议录屏设置

- 分辨率: 1920x1080
- 帧率: 30fps
- 音频: 48kHz 单声道
- 格式: MP4 (H.264)
- 区域: 浏览器窗口（含标题栏）
